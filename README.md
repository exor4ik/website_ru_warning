# PS2 RSOD Warning Page

Страница с предупреждением в стиле Red Screen of Death (RSOD) PlayStation 2.

## Описание

Интерактивная веб-страница с 3D фоном, выполненная в стиле классической ошибки PS2. 

**Особенности:**
- 3D background с использованием Three.js
- Геометрические фигуры (кубы, тетраэдры, октаэдры) в красных оттенках
- Система частиц для создания атмосферы
- Анимированные элементы и эффекты мерцания
- Полностью отзывчивый дизайн (responsive)
- При клике на страницу выполняется редирект на `exor4ik.github.io/website`

## Файлы проекта

- `index.html` - основной HTML файл
- `style.css` - стили с PS2 RSOD эффектами
- `script.js` - Three.js код для 3D сцены и анимации

## Как использовать

1. Откройте `index.html` в браузере
2. Нажмите на страницу для редиректа на основной сайт

## Технологии

- **HTML5** - структура страницы
- **CSS3** - стилизация и анимация
- **JavaScript** - логика и 3D сцена
- **Three.js** - 3D графика

## Браузерная совместимость

- Chrome/Chromium 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Лицензия

Используется для редиректа при ужесточении русских законов о веб-сайтах.
